# 智能食谱推荐功能

## 功能概述

智能食谱推荐功能是基于用户尿酸水平的AI驱动饮食建议系统，通过分析用户的健康状况生成个性化的低嘌呤食谱。

## 主要特性

### 🍽️ 个性化食谱推荐
- 根据用户尿酸水平（正常/偏高/过高）智能推荐
- 支持早餐、午餐、晚餐、加餐分类推荐
- 每个食谱包含详细的食材清单和制作步骤
- 完整的营养分析（热量、蛋白质、脂肪、碳水、纤维、钠）

### 📋 每日菜单规划
- 一日三餐+加餐的完整饮食规划
- 全天营养汇总和健康建议
- 嘌呤风险评估和控制

### 🎯 健康导向设计
- 低嘌呤食材优先选择
- 营养均衡搭配
- 根据健康状况调整推荐策略

## 使用方法

### 1. 访问食谱推荐
1. 登录应用
2. 点击底部导航"趋势"标签
3. 在趋势分析页面下方找到"智能食谱推荐"

### 2. 食谱推荐模式
- **分类推荐**：点击"早餐推荐"、"午餐推荐"、"晚餐推荐"、"加餐推荐"按钮
- **每日菜单**：切换到"每日菜单"标签，点击"生成今日菜单"

### 3. 查看食谱详情
- 点击食谱卡片展开详细信息
- 查看食材清单、制作步骤、营养成分
- 了解嘌呤风险等级和健康建议

## 技术实现

### AI模型集成
- 使用阿里云通义千问API进行食谱生成
- 智能提示词工程确保生成低嘌呤健康食谱
- 支持JSON格式解析和错误重试机制

### 数据结构
```typescript
interface Recipe {
  id: string;
  name: string;
  description: string;
  category: 'breakfast' | 'lunch' | 'dinner' | 'snack';
  ingredients: Ingredient[];
  steps: string[];
  nutrition: NutritionInfo;
  purineScore: 'low' | 'mid' | 'high';
  // ... 更多字段
```

### API接口
- `POST /api/recipe/generate` - 生成单个食谱
- `POST /api/recipe/menu` - 生成每日菜单

## 健康建议

### 尿酸水平分类
- **正常** (<360μmol/L 女性, <420μmol/L 男性)：营养均衡食谱
- **偏高** (360-900μmol/L)：低嘌呤、清淡食谱
- **过高** (>900μmol/L)：严格低嘌呤、降尿酸食谱

### 饮食建议
- 多喝水，每日不少于2000ml
- 避免高嘌呤食物（内脏、海鲜、浓汤等）
- 选择低脂乳制品和新鲜蔬菜
- 控制总热量摄入
- 限制酒精和含糖饮料
